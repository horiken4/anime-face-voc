# anime-face-voc
## Make VOC format dataset by anime face detector

## Usage
```bash
python anime_face_voc.py --listfile=./images.txt --odir=./anime_face_voc
```

`images.txt` is like below:
```txt
/Users/hoge/anime/image1.jpg
/Users/hoge/anime/image2.jpg
...
```
